Express File Upload

A simple and clean file upload application built using Node.js, Express, and Multer.
This project demonstrates how to upload files (such as profile images) from a frontend form and store them on the server.

FEATURES:-
Upload files using an HTML form
Backend powered by Express.js
Uses Multer for handling multipart/form-data
Clean and minimal UI
Beginner-friendly project
Easy to extend and customize

TECH STACK:-
Node.js
Express.js
Multer
HTML
CSS

PROJECT STRUCTURE:-
